API

Uporaba:

API omogoča dostop do zapisov iz podatkovne baze. Lahko uporabite naslednje poti:

1. Pridobivanje vseh zapisov:

/api

Ta pot bo vrnila vse zapise iz baze podatkov v formatu JSON.

2. Pridobivanje zapisov po ID-ju:

/api?id=<celo_stevilo>

Ta pot bo vrnila zapise, katerih ID je večji ali enak vrednosti, ki jo posredujete v parametru 'id'.
