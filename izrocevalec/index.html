Dokumentacija API

Uporaba

API omogoča dostop do zapisov iz podatkovne baze. Na voljo so naslednje poti:

1. Pridobivanje vseh zapisov

/api

Vrne vse zapise iz baze podatkov v formatu JSON.

2. Pridobivanje zapisov po ID-ju

/api?id=<celo_stevilo>

Vrne vse zapise, katerih ID je enak ali večji od posredovane vrednosti.

3. Pridobivanje zapisov glede na čas

/api?t=<število_sekund>

Vrne zapise, ki so bili ustvarjeni v zadnjem določenem številu sekund.
